# Test2D

## Description
一个基于 Godot 4 的 2D 横版过关开源项目。

## Tech Stack
- **Engine**: Godot 4.x
- **Language**: GDScript

## Project Structure
本项目采用模块化的目录结构：

- `ArtAssets/`: 存放美术资源（角色、敌人、环境、UI、特效等）
- `Audio/`: 存放音频资源（背景音乐、音效）
- `Scenes/`: 存放游戏场景和预制体（关卡、角色、敌人、物品、UI）
- `Scripts/`: 存放 GDScript 脚本代码（角色控制、游戏逻辑、自动加载等）
- `Resources/`: 存放游戏资源文件（字体、着色器、主题、图块集）
- `Data/`: 存放配置文件和数据

## Getting Started
1. 克隆本仓库到本地。
2. 打开 Godot Engine (4.x 版本)。
3. 点击 "Import" (导入)。
4. 导航到项目根目录，选择 `project.godot` 文件。
5. 点击 "Import & Edit" (导入并编辑)。
